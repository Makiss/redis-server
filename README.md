# Python Project

A generic Python application.

## Setup

1. Create a virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. Install the project in development mode:
   ```
   pip install -e ".[dev]"
   ```
